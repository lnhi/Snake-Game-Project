## Hướng dẫn cài đặt
- Tải source code về máy.  
- Mở folder code trong **_VSCode_**.  
- Mở termial.  
- Để **chạy game**, sử dụng _câu lệnh_ sau:  
```C++
mingw32-make && .\main.exe
```
- Nội dung gì đó nhắc đến `keyword`
  - Nội dung tab 2
    - Nội dung tab 3
